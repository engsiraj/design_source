The Design Source
For developers by developer.

Using a user-friendly user interface design system, you can customize your design

Typography | Colors | Grid | Width | Responsive | Elements | Visible | Hover | Shadows | Media

About Design source design system

The Design Source is a User Interface Design System built in html/css that allows resources to be reused to build consistent user interfaces and brands

An open source project developed by [ Siraj ], a React frontend developer and Design system engineer.

Get started

For fast-paced design, you can use a CDN or starter file as index.html

CDN

<link href='https://cdn.jsdelivr.net/gh/sirajofcl
/designsystem@0.1.1/style.css' rel="stylesheet">
